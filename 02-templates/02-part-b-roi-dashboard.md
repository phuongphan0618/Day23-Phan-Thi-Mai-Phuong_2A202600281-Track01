## Gợi Ý Lớp Đo

| Lớp đo | Câu hỏi | Ví dụ |
|---|---|---|
| Activation | Người dùng hoàn thành task đầu tiên chưa? | % users complete first AI-assisted task |
| Engagement | Có quay lại dùng đúng nhịp không? | % repeat weekly / per session |
| Productivity | Công việc nhanh hơn hoặc nhiều hơn không? | task time, throughput |
| Quality | Output có ít lỗi / ít rework không? | error rate, acceptance rate, QA score |
| Trust | Người dùng accept, override, escalate thế nào? | override rate, escalation rate, CSAT |
| Value | Có tạo kết quả kinh doanh / học tập / vận hành không? | cost saved, SLA, NPS, learning outcome |

## Các Lỗi Cần Tránh

- Chỉ đo login, active users, prompt count.
- Không có baseline.
- Data source ghi chung chung: "analytics", "survey", "feedback".
- Người phụ trách ghi "team" thay vì một role cụ thể.
- Productivity tăng nhưng không có Quality / Trust đi kèm.
